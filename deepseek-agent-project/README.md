# DeepSeek Agent Project

A new project for DeepSeek agent development, based on the provided interface screenshots.

## Project Structure

```
deepseek-agent-project/
├── README.md              # Project documentation
├── package.json           # Node.js dependencies
├── src/
│   ├── main.js            # Main application entry
│   ├── components/        # React components
│   │   ├── ChatWindow.js  # Chat window component
│   │   ├── Settings.js    # Settings component
│   │   └── Sidebar.js     # Sidebar navigation
│   ├── pages/             # Application pages
│   │   ├── Chat.js        # Chat page
│   │   └── SettingsPage.js # Settings page
│   └── utils/             # Utility functions
└── public/                # Static assets
    └── index.html         # HTML template
```

## Features

- DeepSeek chat interface
- System settings with theme options
- Account management
- Data management
- Service agreement

## Installation

```bash
npm install
```

## Usage

```bash
npm start
```

## Development

This project is based on the DeepSeek web interface screenshots provided. It includes the main components:

1. Chat window with message input
2. Sidebar navigation
3. System settings with:
   - Theme options (light, dark, follow system)
   - Language settings
   - Account management
   - Data management
   - Service agreement
