## 1. UniMCP使用指南

# UniMCP使用指南 - MCP万能聚合器

## 1. 工具概述

**UniMCP** 是一个革命性的MCP聚合工具，将多个MCP工具统一整合为单一、完美的集成系统。它支持自动发现、智能路由、统一接口调用，可以被Trae CN安全调用。

### 1.1 核心功能

- **自动发现**：自动发现系统中的所有MCP工具
- **智能路由**：根据命令自动路由到正确的MCP工具
- **统一接口**：提供单一的接口调用所有MCP工具
- **MCP合并**：将多个MCP工具融合为单一的MCP工具
- **错误修复**：自动修复MCP工具的各种错误
- **工具分析**：详细分析MCP工具的结构和配置
- **安全调用**：支持Trae CN安全调用

### 1.2 适用场景

- 管理大量MCP工具
- 将多个相关MCP工具融合为单一工具
- 修复损坏的MCP工具
- 分析MCP工具的结构和配置
- 统一接口调用多个MCP工具

## 2. 快速开始

### 2.1 Python API调用


## 2. Windows激活指南

# Windows 系统合法激活指南

## 重要提示
激活 Windows 系统的唯一合法且安全的方式是通过 Microsoft 官方渠道获取授权。使用非官方方法（如破解工具、密钥生成器等）不仅违反 Microsoft 服务条款，还可能带来安全风险（如恶意软件、系统不稳定等）。

## 合法激活方法

### 1. 购买官方许可证密钥
- **Microsoft 官网**：直接从 Microsoft Store 购买 Windows 许可证
- **授权零售商**：从授权的电子产品零售商处购买正版密钥
- **预装系统**：购买新电脑时选择预装正版 Windows 系统的设备

### 2. 使用数字许可证激活
如果您的电脑曾经激活过相同版本的 Windows，并且使用 Microsoft 账户登录，系统可能会自动使用数字许可证激活。

### 3. 批量激活（针对企业/组织）
- 批量激活服务 (KMS)
- 多次激活密钥 (MAK)

## 如何激活 Windows

1. **打开设置**：点击开始菜单 → 设置 → 更新和安全 → 激活
2. **选择激活方式**：
   - 如果有密钥：选择「更改产品密钥」，输入您的正版密钥
   - 如果是数字许可证：确保已登录 Microsoft 账户，系统会自动激活
   - 如果需要购买：点击「转到 Microsoft Store」购买正版密钥

## 验证激活状态
在「设置 → 更新和安全 → 激活」页面，您可以查看 Windows 的激活状态和许可证类型。

## 注意事项
- 请勿从非官方渠道购买低价密钥，这些密钥可能是被盗取或批量生成的，容易被封禁
- 不要使用任何破解工具或激活脚本，这些工具可能包含恶意软件
- 定期更新 Windows 系统以确保安全和稳定性

通过合法渠道激活 Windows，您将获得完整的系统功能、安全更新和 Microsoft 官方支持。

## 3. Windows Activation Manual Guide

# Windows Activation Manual Guide

## Important Notice
The only legal and safe way to activate Windows is through official Microsoft channels. Using unauthorized methods (like cracks, key generators, etc.) violates Microsoft's terms of service and may introduce security risks.

## Legal Activation Methods

### 1. Purchase Official License Key
- **Microsoft Official Website**: Buy Windows license directly from Microsoft Store
- **Authorized Retailers**: Purchase genuine keys from authorized electronics retailers
- **Pre-installed Systems**: Buy new computers with pre-installed genuine Windows

### 2. Digital License Activation
If your PC has previously activated the same Windows version and you're signed in with a Microsoft account, the system may automatically activate using a digital license.

### 3. Volume Activation (for enterprises/organizations)
- Key Management Service (KMS)
- Multiple Activation Key (MAK)

## Step-by-Step Manual Activation

### Method 1: Activate with Product Key
1. **Open Settings**: Click Start Menu → Settings → Update & Security → Activation
2. **Enter Product Key**: 
   - Click "Change product key"
   - Enter your genuine product key (25 characters)
   - Follow the on-screen instructions

### Method 2: Activate with Digital License
1. **Ensure Internet Connection**: Make sure your PC is connected to the internet
2. **Sign in with Microsoft Account**: Go to Settings → Accounts → Sign in with a Microsoft account
3. **Automatic Activation**: Windows should automatically activate within a few minutes

### Method 3: Command Prompt Activation
1. **Open Command Prompt as Administrator**:
   - Press Windows + X → Select "Command Prompt (Admin)" or "Windows PowerShell (Admin)"

2. **Check Current Activation Status**:
   ```
   slmgr /dli
   ```

3. **Uninstall Current Key (if needed)**:
   ```
   slmgr /upk
   ```

4. **Install New Product Key**:
   ```
   slmgr /ipk YOUR-PRODUCT-KEY-HERE
   ```
   *Replace YOUR-PRODUCT-KEY-HERE with your genuine 25-character key*

5. **Activate Windows**:
   ```
   slmgr /ato
   ```

6. **Verify Activation**:
   ```
   slmgr /dli
   ```

### Method 4: Fix Common Activation Errors

#### Error 0xC004F074 (KMS Activation Failed)
1. **Sync System Time**:
   ```
   w32tm /resync /force
   ```

2. **Check KMS Server Connection**:
   ```
   slmgr /skms kms.microsoft.com:1688
   slmgr /ato
   ```

3. **Reset Software Protection Service**:
   ```
   net stop sppsvc
   cd %windir%\ServiceProfiles\NetworkService\AppData\Roaming\Microsoft\SoftwareProtectionPlatform
   del tokens.dat
   net start sppsvc
   slmgr /ato
   ```

#### Error 0xC004C003 (Activation Server Determined the Specified Product Key is Blocked)
- This usually means the product key is invalid or blocked. Contact Microsoft Support for assistance.

## Verify Activation Status
- **Settings**: Go to Settings → Update & Security → Activation
- **Command Prompt**: Run `slmgr /dli` or `slmgr /xpr`

## Best Practices
- Always keep your Windows system updated
- Use a Microsoft account to link your digital license
- Store your product key in a safe place
- Avoid sharing your product key with others

## Microsoft Support
If you encounter persistent activation issues, contact Microsoft Support for assistance:
- [Microsoft Support Website](https://support.microsoft.com/)
- Phone: Check your region's support number on the Microsoft website

Remember: Only genuine Windows activation provides full system functionality, security updates, and official support!

## 5. 安全机制

### 5.1 速率限制

默认限制为60请求/分钟，可通过配置文件调整。当请求速率超过限制时，工具会返回错误信息，提示用户稍后再试。

### 5.2 命令白名单

只允许执行配置文件中指定的命令，默认允许所有命令。可通过配置文件设置白名单：

```json
{
    "security": {
        "command_whitelist": ["get_file_info", "list_mcp_tools", "convert_url_to_mcp"]
    }
}
```

### 5.3 缓存机制

默认缓存时间为300秒（5分钟），可通过配置文件调整。缓存机制可提高重复请求的响应速度。

### 5.4 输入验证

所有输入参数都会经过验证，确保符合预期格式和范围。
